# Quiz Filosofico

Quiz interattivo per scoprire a quale filosofo ti avvicini di più.

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS 4
- JavaScript ES6+

## Funzionalità

- 5 domande con risposte multiple
- Calcolo del risultato basato sui punteggi accumulati
- Schermata introduttiva e risultato finale
- Navigazione avanti/indietro tra le domande
- Dark mode automatica (segue le preferenze del sistema)

## Filosofi

- Platone
- Nietzsche
- Schopenhauer

## Installazione
```bash
npm install
npm run dev
```
